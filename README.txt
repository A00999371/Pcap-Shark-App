Pcap Wireshark Java Project
Authors:
Mark Skerl, Amrit Gill and Creighton Lee

About:
-This is a pcap file reading program which can be used to view different types of data from pcap internet files. 

Notes:
-This program requires Java to run. https://www.java.com/en/
-The file pcapProject.jar will only run if the file jnetpcap.dll is in the same folder.  

Instructions:
-Double click pcapProject.jar to run.
-Select File -> Open File, from the top left.
-Navigate to the directory which the program is in, then select and open test_pcap_file.pcap.
